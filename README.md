# MAZE by Щёлочь — Telegram bot

Минимальный онбординг-бот для игры MAZE.

## Что делает

При команде `/start` присылает картинку, описание игры и кнопку «🎮 Играть»,
открывающую Mini App во весь экран Telegram.

На любое другое сообщение отвечает «Жми «Играть» внизу» с той же кнопкой.

## Запуск

Бот работает на бесплатном плане Render.com через long polling.

### Переменные окружения

- `BOT_TOKEN` — токен бота из @BotFather (обязательно)
- `WEB_APP_URL` — URL Mini App (по умолчанию: Netlify-ссылка игры)

### Зависимости

```
python-telegram-bot==22.7
```
